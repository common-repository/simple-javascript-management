== Description ==

This plugin allows you to manage all of your JavaScript files through the WordPress administration interface instead of poking in the code to add them.

Use of Simple JavaScript without management

You do not need to use the Editor to manage your JavaScript files

You can change the order of the JavaScript files are loaded 

You can have multiple JavaScript files standby but that they used on the page.

You can choose where to load JavaScript files, in witch page and/or body or head tag.

Quite simply, easier way to handle JavaScript files.

Just activate the plugin and paste this line of code in to your head tag of your WordPress theme.

== Installation ==

1. Upload `sjsm` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
